摘要：
我们这此选择了中国实时碳数据折线图来进行可视化分析。二氧化碳（CO2）是导致全球气候变暖的主要温室气体。
我们分析了从年初COVID-19在中国爆发一直到疫情在中国逐渐得到控制这段时间中国的二氧化碳排放量与去年同时期碳排放量的对比折线图。
我们认为分析这张图是非常有意义的。碳监测是一项国际倡议，定期更新碳监测数据将使我们更好地了解政策行动和经济转变、能源价格、假期、周末和天气如何控制二氧化碳排放的动态。
为了更好的研究疫情对二氧化碳排放量的影响，我们分析了图中的数据类型，视觉变量等，又分析了相关的认知理论，还发现了图中的一些可视化问题并做出了修改，让这张折线图更加符合信息可视化的规范要求。
